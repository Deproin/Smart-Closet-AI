# 🚀 Project Portfolio | معرض الأعمال البرمجي

This document provides a technical overview of the major projects developed, highlighting the tech stack and core functionalities.

---

## 🎙️ Nobdah (نبضة) - Random Voice Chat
*A fast-paced, privacy-focused random voice communication app.*
*   **Frontend:** Flutter (Mobile).
*   **Backend:** Python (Custom WebSockets), Socket.io.
*   **Real-time Audio:** WebRTC for high-quality, low-latency voice calls.
*   **Key Features:** Instant anonymous pairing, high-level encryption, real-time connection management.

## 🚛 Waitek (وايتك) - Logistics & Water Delivery
*Comprehensive platform for managing water delivery and truck tracking.*
*   **Frontend:** Flutter (State management with Bloc/Provider).
*   **Backend:** FastAPI (Python) - High performance asynchronous API.
*   **Database:** PostgreSQL for robust order and user data management.
*   **Tracking:** Google Maps API integration for real-time truck location and driver routing.
*   **Key Features:** Driver availability status, real-time order tracking, logistics optimization.

## 👗 Smart Closet AI - Intelligent Wardrobe Manager
*AI-driven system for digitizing and managing personal fashion.*
*   **Web Frontend:** React.js + Vite + Tailwind CSS (Premium UI with Dark Mode).
*   **Mobile Frontend:** Flutter (Bilingual support AR/EN).
*   **Backend:** FastAPI (Python), PostgreSQL, SQLAlchemy.
*   **AI Engine:** YOLO/CLIP models for automatic clothing classification and color detection.
*   **DevOps:** GitHub Actions for automated CI/CD (Auto-building APKs).
*   **Integration:** Real-time Weather API (OpenWeatherMap) for smart outfit suggestions.

## 💼 Kifah (كفاح) - AI Recruitment Platform
*Smart matching system between job seekers and employers.*
*   **Backend:** PHP (Laravel Framework), MySQL.
*   **AI Logic:** Custom matching algorithms for CV-to-Job requirement analysis.
*   **Key Features:** Automated candidate filtering, instant job-matching scores.

## 🏥 Medical Clinic Management System
*Desktop management software for clinics and patient records.*
*   **Language:** C# (#.NET Framework).
*   **Architecture:** Desktop-based with secure local/cloud database sync.
*   **Key Features:** Patient appointment scheduling, medical history archiving, advanced role-based access control (RBAC).

## 🦷 Teeth Health Monitoring Website
*Interactive web portal for monitoring dental status and hygiene.*
*   **Tech Stack:** Semantic HTML5, CSS3 (Custom Flex/Grid), Vanilla JavaScript.
*   **UI/UX:** Focused on accessibility and ease of use for medical tracking.
